# AI Configuration\n\nThis repository contains configuration files for various AI tools including GitHub Copilot CLI.\n\n## Setup with GNU Stow\n\nTo sync configurations to your home directory:\n\n```bash\ncd ~/personal/ai-tooling\nstow -t ~ .\n```\n\nThis will create symlinks from ~/.copilot to this repository's .copilot directory.\n\n## Structure\n\n- `.copilot/agents/` - User-level custom agents for Copilot CLI\n
